# wipry_raw

A command-line utility to dump spectral data from a WiPry Clarity spectrum analyzer.

```
Usage: wipry_raw [options]
Options:
  -l, --list-ranges    List supported ranges.
  -r, --range RANGE    Use RANGE.
  -h, --help           Print this message and exit.
```
